# üìö Course Structure

The Professional Certificate includes 9 structured courses, progressing from foundational concepts to advanced project implementation.

---

## Course 1: Introduction to Data Analytics

**Focus:** Understanding the Data Analytics process

**Key Learnings:**

- What Data Analytics is
- Steps in the analytics process
- Different data roles (Data Analyst, Data Scientist, Data Engineer, etc.)
- Types of data, file formats, and repositories
- Basic business case analysis

---

## Course 2: Excel Basics for Data Analysis

**Focus:** Spreadsheet-based data analysis

**Key Learnings:**

- Spreadsheet fundamentals
- Data entry and formula usage
- Data cleaning techniques
- Filtering and sorting
- Lookup functions
- Pivot table analysis

---

## Course 3: Data Visualization & Dashboards (Excel and Cognos)

**Focus:** Data storytelling through visualization

**Key Learnings:**

- Creating charts and pivot charts
- Data storytelling principles
- Advanced Excel visualizations
- Dashboard creation
- Working with IBM Cognos Analytics

---

## Course 4: Python for Data Science

**Focus:** Programming for data analysis

**Key Learnings:**

- Python basics and data types
- Lists, tuples, loops, and conditional statements
- Working with Pandas and NumPy
- Web scraping using Beautiful Soup
- Accessing APIs using Jupyter Notebooks

---

## Course 5: Python Project for Data Science

**Focus:** Applying Python in a real-world project

**Key Learnings:**

- Data extraction and cleaning
- Data visualization using Python
- Dashboard creation
- Working with real datasets
- Project documentation

## Course 6: Databases and SQL for Data Science

**Focus:** Database querying and management

**Key Learnings:**

- Writing SELECT, INSERT, UPDATE, DELETE statements

- Working with JOIN operations
- Creating and managing tables
- Accessing cloud databases
- Integrating SQL with Python

**Focus:** Exploratory Data Analysis and Modeling

**Key Learnings:**
